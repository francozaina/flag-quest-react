# 🚩 Flag Quest: Interactive Learning Game

An engaging, fast-paced flag guessing game built with **React**. This project showcases solid fundamentals in component-based architecture, state management, and interactive UI design, developed during my technical training at **ORT Argentina**.

---

## 🚀 Live Demo
[https://flag-quest-react.vercel.app/]

---

## ✨ Key Features
* **Dynamic Gameplay:** Randomly generated flag challenges to test geographic knowledge.
* **Real-time Scoring:** Instant feedback logic for correct and incorrect answers.
* **Component Architecture:** Modularized React components for better maintainability.
* **Responsive Design:** Fully functional across desktop and mobile devices.

## 🛠️ Tech Stack
* **Frontend:** React.js (Hooks: `useState`, `useEffect`)
* **Styling:** CSS3 (Flexbox/Grid)
* **Environment:** Node.js & NPM
* **Version Control:** Git & GitHub

## 🔧 Installation & Setup
To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/francozaina/flag-quest-react
   ```

2. **Navigate to the project folder:**
   ```bash
   cd flag-quest-react
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm start
   ```
   The app will run at `http://localhost:3000`.
