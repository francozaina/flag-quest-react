# 🚩 Flag Quest React

Aplicación web desarrollada con **React**, orientada al reconocimiento de banderas mediante una dinámica interactiva de juego, consumiendo información desde una **API externa**.

El proyecto fue realizado originalmente como trabajo académico y luego adaptado/mejorado para dejarlo más prolijo, presentable y fácil de mostrar como parte de mi portfolio.

---

## 🚀 Demo

**Live Demo:** https://flag-quest-react.vercel.app/

## 🛠️ Tecnologías utilizadas

- **React**
- **JavaScript**
- **HTML5**
- **CSS3**
- **Axios**
- **Node.js**
- **NPM**

---

## 📌 Características principales

- Juego interactivo de reconocimiento de banderas
- Consumo de datos desde una **API externa**
- Interfaz web con renderizado dinámico
- Lógica de juego del lado del cliente
- Componentes reutilizables en React
- Diseño responsive para distintos dispositivos

---

## 🧠 Objetivo del proyecto

El objetivo principal fue practicar conceptos de desarrollo **frontend**, trabajando sobre:

- Construcción de interfaces web
- Organización de componentes
- Consumo de APIs y manejo de datos externos
- Lógica de interacción del lado del cliente
- Estructuración de una aplicación con React

---

## 📂 Estructura general

El proyecto está organizado de forma modular, siguiendo una estructura basada en componentes:

- **Components** → componentes reutilizables de interfaz
- **App / estructura principal** → organización general de la aplicación
- **State Management** → lógica de juego e interacción
- **API Integration** → obtención y uso de datos externos
- **Styles** → estilos visuales y responsive design

## 🔧 Instalación y Configuración Local

Para correr este proyecto en tu entorno de desarrollo local seguí estos pasos:

### 1. Clonar el repositorio
```bash
git clone https://github.com/francozaina/flag-quest-react.git
```

### 2. Ingresar a la carpeta del proyecto
```bash
cd flag-quest-react
```

### 3. Instalar dependencias
```bash
npm install
```

### 4. Ejecutar el proyecto
```bash
npm start
```

La aplicación debería ejecutarse en:

```bash
http://localhost:3000
```
