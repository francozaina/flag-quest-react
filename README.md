- [cite_start]Interactive flag recognition game logic[cite: 27].

### 🧠 Project Goals
The main focus was to practice **frontend** development concepts:
- Building intuitive web interfaces.
- Component-based architecture and organization.
- API integration and handling external asynchronous data.
- Implementing complex interaction logic on the client side.

### 📂 Structure
- **Components**: Reusable UI elements.
- **App / Core**: General application orchestration.
- **State Management**: Game flow and user interaction logic.
- **API Integration**: Fetching and processing external data.
- **Styles**: Visual design and responsive layouts.

### 🔧 Local Setup
1. **Clone the repository:** `git clone https://github.com/francozaina/flag-quest-react.git`
2. **Navigate to the folder:** `cd flag-quest-react`
3. **Install dependencies:** `npm install`
4. **Run the project:** `npm start` (Runs on `http://localhost:3000`)
